# LED_shines

LED control using Arduino UNO
This is a beginner robotics project using Arduino.

The project makes an LED blink every second using Arduino Uno.
LED turns ON and OFF every second.

Components required:
- Arduino Uno
- LED
- Breadboard
- Jumper wires
- USB cable

Working:
The Arduino sends HIGH and LOW signals to pin 13, which turns the LED ON and OFF.
